Application functionality:

Piskel clone is an application based on Canvas API. There are three 32х32 / 64х64 / 128х128 user-defined units of canvas size. Four tools are available for using:

  • pen for drawing on canvas;
  • bucket for painting all pixels of the same color;
  • eraser for cleaning canvas;
  • stroke for drawing straight lines.
Also user can draw with two different colors by clicking on left (primary color) or right (secondary color) mouse buttons. There is an ability to export the final result to the local file system in a gif format.
